# Travel Website Project

A travel website designed to increase tourism to the fictional island of Taniti. Built as a UX design project focused on user experience and conversion. All photos used came from Unsplash.

## Live Demo
[View Site](https://cabra12.github.io/travel-website-frontend-ux/)

## Built With
- HTML
- CSS
- Javascript

## Features
- built responsive features with CSS elements like grid, media query, and clamp
- website will be responsive to desktop, tablet and phone screens
- used Javascript to simulate a form being sent
- animated cards with CSS
